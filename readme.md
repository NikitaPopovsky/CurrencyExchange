₽ → $ Currency Exchange Service
REST API + Web UI для управления валютами и курсами обмена
✅ Jakarta Servlet 6 (Tomcat 11) • ✅ PostgreSQL • ✅ Bootstrap 5 • ✅ jQuery

🌟 Основные возможности

| Функция | Описание |
|---------|---------|
| Управление валютами | Добавление новых валют (код, название, символ) |
| Управление курсами | Установка и редактирование курсов обмена между валютами |
| Конвертация | Мгновенный расчёт суммы в целевой валюте |
| REST API | Полноценный JSON API для интеграции |
| Веб-интерфейс | Интуитивный UI с таблицами, формами и модальными окнами |

🛠 Технологии

| Категория | Стек                                               |
|-----------|----------------------------------------------------|
| Backend | Jakarta EE 10, Servlet API 6, Tomcat 11.0.15       |
| База данных | PostgreSQL, JDBC                                   |
| Frontend | HTML5, Bootstrap 5, jQuery 3.6.3                   |
| Сборка | Maven, war-артефакт                                |
| Развёртывание | Облачный сервер (Ubuntu 24.04) |

🚀 Быстрый старт (локально)

**Требования**
- Java 21+
- Maven 3.8+
- Tomcat 11 (или встроенный через IDE)

**Установка**

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ваш-логин/CurrencyExchange.git
cd CurrencyExchange

# 2. Соберите проект
mvn clean package

# 3. Запустите Tomcat и задеплойте ROOT.war
#    (или запустите через IntelliJ IDEA: Run → Tomcat)
